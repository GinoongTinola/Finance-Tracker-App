# 💸 Financial Roadmap

A personal monthly budget tracker for Filipino earners — semi-monthly (cutoff) basis, with Maya Bank savings projection.

**[→ Open App](https://GinoongTinola.github.io/Finance-Tracker-App)**

---

## Features

- 📋 **Monthly Table** — log income & expenses per cutoff (1–15 / 16–end), check ✓ to confirm
- 📊 **Overview** — yearly summary, unallocated cash, savings goal progress
- 📈 **Charts** — income vs expenses, expense breakdown, goals progress
- 💰 **Savings Projection** — Maya Bank tiered interest with exact daily accrual formula
- 🗓 **Multi-year** — supports 2020–2040
- 💾 **Export / Import JSON** — backup and restore anytime
- 📱 **Mobile-ready** — bottom nav, APK-compatible via WebIntoApp

---

## Data Transfer (PC → Phone)

Export JSON → send via Telegram / GDrive → Import JSON on phone.
